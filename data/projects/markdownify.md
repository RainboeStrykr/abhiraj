The only markdown editor you will ever need.

## Features

1. **Save Your Files**: You can save your typed docs in local storage

2. **Download Files**: Download your markdown docs as in .md format or in HTML format.

3. **Live Preview**: You can see the live preview of the markdown you are typing for better accuracy and accessiblity.

4. **Light/Dark Mode**: Oh yeah, you got it!

5. **Auto-spellcheck**: Corrects and provides the right grammar and spelling when turned on so you never make another mistake.
