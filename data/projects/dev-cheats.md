Dev-Cheats is an open-source repository of beginner-friendly cheatsheets for popular languages. I created this project as a go-to reference point while I am practicing problems. Hopefully, this will fulfil the same purpose for all the students who are learning coding. The best process which I've found when you're learning a new language is to make notes. One can take help from this website to do so. As of now, it has cheatsheets for Python, Java, C++, Javascript, HTML and CSS. I will be adding more languages in the future as well.
