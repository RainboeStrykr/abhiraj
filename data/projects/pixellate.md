Pixellate is a pixel-art editor created with HTML canvas and Javascript. It is a Progressive Web App (PWA) with offline compatibility. It is mobile-friendly and is very easy to use.

## Features

- Wide variety of tools ranging from image to pixel art converter to paint tools, all for you to create the best pixel art.
- PIxellate is a progressive web app which can be downloaded on the mobile device or the pc and be used when offline.
- We are completely free and open source. Feel free to contribute and support us by starring our github repository
